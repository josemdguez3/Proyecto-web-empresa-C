Para añadir proyecto a github: 1º se crea un nuevo repositorio en github, se copia la url del repositorio, vamos a vscode y en view, en la paleta d comandos escribimos git clone y pegamos la url. Pulsamos enter y nos abre en nuestro equipo para seleccionar o crear carpeta seleccionamos la carpeta y se nos abrirá en vscode. AL realizar cambios y guardar iremos a el 3er dibujo en la columna haremos un commit dentro de "Mensaje", escribiendo (comienzo o cambio) seguido de control + enter, a continuación, en los tres puntos presionaremos (Insertar que equivale a push) y tendremos la información clonada en github Desde github podremos modificar el contenido guardando al presionar commit changes y refrescando luego en vscode en el circulo de flechas situado abajo izq.
